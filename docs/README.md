## 🎯기능 구현 목록

- **사용자 에게 자동차 이름 입력 받기**
    - [X] 자동차 이름 입력 받기
    - [X] 자동차 이름 쉼표(,)를 기준으로 구분
    - [X] 이름은 공백 제외 5자 이하만 가능
        - **입력 후 예외 처리**
            - [X] 자동차 이름 중복되는 경우
            - [X] 자동차 이름 NULL 혹은 공백일 경우
            - [X] 자동차 이름 5자 넘는지 확인

- **사용자 에게 이동 횟수 입력 받기**
    - [X] 이동 횟수 입력 받기
    - [X] 입력값 21억 까지 입력 가능
        - **입력 전 예외 처리**
            - [X] 입력값 정수인지 확인
            - [X] 입력값 NULL 혹은 공백일 경우
        - **입력 후 예외 처리**
            - [X] 입력값 음수인 경우

- **자동차 전진 조건**
    - [X] 경주 할 자동차에 대한 0~9 사이의 랜덤 숫자 생성
    - [X] 생성된 랜덤 숫자 4 이상일 경우 전진

- **결과 출력**
    - [X] 가장 많이 이동한 자동차의 이름 출력
    - [X] 우승자가 여러명일 경우 쉼표(,)를 이용하여 구분
